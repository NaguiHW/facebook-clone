## Getting started

To get started with the app, clone the repo and then install the needed gems:

```
$ bundle install --without production
```

Next, create the database:

```
$ rails db:create
```

Next, migrate the database:

```
$ rails db:migrate
```

Ready to run the app in a local server:

```
$ rails server
```
